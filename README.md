# pychip8
Chip-8 emulator with python

## Requirements

- Python 3.10+
- Pygame 2.1+

## Installation

```bash
  git clone git@github.com:ACSouzaJr/pychip8.git
  cd pychip8
  pip install -r requirements.txt
 ```
 
 ## Run it
 
 ```bash
 python app.py --rom <path-to-rom>
 ```
